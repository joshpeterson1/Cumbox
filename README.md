Cumbox

Installation

Clone the repository:

git clone https://github.com/joshpeterson1/Cumbox

Go into the directory:

cd cumbox

Run The appropriate Installer (arm, arm64, i486):

./cumscriptPi.sh
./cumscriptArm64.sh
./cumscriptIntel.sh
